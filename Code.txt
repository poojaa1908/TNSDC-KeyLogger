1.	Go to your GitHub repository.
	2.	Click on the “Pull requests” tab.
	3.	Click on the “New pull request” button.
	4.	At the top, you’ll see:
Change base to the branch with the older version (e.g., main).
	•	Change compare to the branch with the newer version (e.g., version2).

5.	GitHub will show a full file-by-file diff:
	•	✅ Added files
	•	❌ Deleted files
	•	✏️ Modified lines in existing files
